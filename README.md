# node-red-contrib-zigbee2mqtt
Node-Red Nodes for Zigbee2mqtt connectivity.

Available nodes are:
* wb-input: A node to subscribe to zigbee2mqtt mqtt topics
* wb-output: Publish messages to zigbee2mqtt devices
